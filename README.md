# Reflex game
## Test your reflexes

_Poles are ascending and the player needs to press the right button to put them down._

## Controls

| Button        | Function                |
| :-----------: |:-------------:          |
| G             | Start                   |
| S             | Stop                    |
| R             | Restart                 |
| 1-9           | Put down a pole         |
| ESC           | Exit                    |
